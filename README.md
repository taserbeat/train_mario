# train_mario

Source:  
https://github.com/tsunaki00/super_mario.git

# Requires

- Python >= 3.6.0
- [fceux](http://fceux.com/web/home.html)
  - Add `fceux` to `PATH`

# How to use it

```
bash reset_venv.sh
python start.py
```
